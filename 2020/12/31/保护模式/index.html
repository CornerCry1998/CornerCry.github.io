一、32位段寄存器

例: ES寻址

ES 等于 0023

等于二进制:

0000000000100011

拆分后

0000000000100 0 11

得到00cff300`0000ffff,对照上面的结构图

BASE = 0

LIMIT = 0xfffff

Attr = (0)cf3 (一共占2个字节)

Attr.G = 1 (粒度 = 1 代表 (0xfffff+1) * 4096 - 1) = 0xffffffff (粒度 = 0 LIMIT补0 = 0x000fffff

Attr.D = 1

Attr.P = 1

Attr.DPL = 11

Attr.type = 0011
